
## Kiosk


#### 자바와 oracle를 활용한 자바 미니 프로젝트입니다.
#### 구조는 ***mvc***구조로 짰고, 프론트 부분은 ***GUI***를 활용하여 
#### 직관적인 키오스크 ***UI***를 구현해보았고, 프론트에 데이터를
#### 연결해주고 넘겨주는 백에서의 서비스 작업도 해보았습니다. 
#### 벤치마킹 사이트 : <https://www.burgerking.co.kr/>
